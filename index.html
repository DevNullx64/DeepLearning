DCatcher's Deep Learning Library

Simple and Cutting-edge Deep Learning Library accelerated with GPU using C++ AMP

Download this project as a .zip file Download this project as a tar.gz file

This project provides the simplest deep learning library with GPU acceleration in pure C++11.

Unique features of this projects:

  1. Only one .h/.cpp file for the core logic and model. To keep everything as simple as possible, I only implement the most successful model and training methods. So it's good place for the beginner of deep learning.

  2. Following the latest update in deep learning research. A good place for deep learning veteran to try out your own ideas based on the latest work.

  3. Support GPU acceleration via c++ amp. So it inherits all the advantages of amp:

    • support AMD, NVIDIA and INTEL video cards as long as your video card supports directx11.
    • pure c++11 syntax and fully object-oriented, no need for c-style vender-specific kernel, easy to understand.
    • easy GPU debugging with visual studio

    With GPU acceleration, you can solve real problem with a single machine.
